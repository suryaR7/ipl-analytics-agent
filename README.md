🏏 IPL Analytics AI – Natural Language to SQL Engine
🚀 Overview

An AI-powered analytics platform that converts natural language questions into SQL queries over IPL ball-by-ball data and dynamically visualizes results.

🧠 Architecture

User → Streamlit UI → LLM → SQL Generator → Validator → SQLite → Pandas → Plotly

⚙️ Tech Stack

Python

SQLite

Streamlit

Groq LLM

Pandas

Plotly

📊 Features

Natural language query support

Player comparison logic

Strike rate / economy calculations

Duck detection

Secure SQL validation

Interactive visualizations

Authentication system
